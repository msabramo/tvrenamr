Tv Renamr

Tv Renamr is a command line utility to renamr TV show files into a cleaner format written in Python by George Hickman.

The source can be found on Github along with the issue tracker.

Docs are currently confined to the readme on Github, but should progress over to Read the Docs soon.

Install via pip or easy install from the PyPI.

Run on the command line with tvr and a filename, directory, or wildcard (*.avi).
